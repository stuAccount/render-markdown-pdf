# 实验三 PDF 渲染 Smoke Test

## 一 检查项

- 深层标题应当独立成行，而不是和下一段文字挤在同一行。
- 表格列宽应根据内容分配，而不是把 IP、端口和备注列强制挤成等宽。
- 普通中文/CJK 文档应能正常渲染列表、代码块、图片和表格。

#### 深层标题检查

这一段用于 smoke test，确认 `####` 标题之后的正文在 PDF 中会另起一段。

tcpdump -i en0 -nn host 93.184.216.34 and tcp
